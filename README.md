# Secure Notes Web Application

## Overview

This project was developed for the course CT-477 Secure Software Design & Development at NED University of Engineering & Technology.

The project demonstrates common web application vulnerabilities and their secure remediation using a Flask-based Notes Management application.

The application contains:
- Vulnerable Version
- Secure Fixed Version
- Automated Security Testing


## Implemented Vulnerabilities

- SQL Injection
- Cross-Site Scripting (XSS)
- Weak Password Storage



## Security Fixes

- Parameterized Queries
- Password Hashing
- Output Escaping
- Secure Input Handling



## Technologies Used

- Python
- Flask
- SQLite
- Pytest



## Running Vulnerable Application

cd vulnerable_app

python3 app.py


## Running Secure Application

cd fixed_app

python3 app.py



## Running Tests

cd tests

pytest -v
